🚀 Entro - Company Landing Page

A modern and responsive Company Landing Page built using PHP, Bootstrap 5, MySQL, and JavaScript. The project includes a clean business website, a Blog CMS, a Contact Form, and a secure Admin Dashboard for managing blog posts and customer inquiries.

🌐 Live Demo

Website: https://hriday.infinityfree.me

🚀 Installation

1. Clone the repository:
   
   git clone https://github.com/your-username/entro.git

2. Move the project folder to the "htdocs" directory in XAMPP.

3. Start Apache and MySQL.

4. Import the SQL database into phpMyAdmin.

5. Update your database credentials in the configuration file.

6. Open your browser and visit:
   
   http://localhost/entro
